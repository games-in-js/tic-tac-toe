# TIC TAC TOE (JOGO DA VELHA)

![Tic tac toe](/src/docs/tic-tac-toe.webp)

Aprenda a criar um jogo da velha funcional e interativo.

## Stack

- Vite
- React
- Typescript
- Tailwind
- Motion

```
npm install
npm run dev
```
